#Info

  De lagrede vinnerene er lagret i JSON filen. Dersom den ikke eksistere blir
  den laget når det trenges. 

  Listen av elever som er gyldig til trekkning ligger i CSV filen. Dersom denne
  også ikke eksisterer, blir den laget ved data hentet direkte fra IT-forlaget.

  Hvis en ønsker å resette hele programmet (fjerne alle lagrede vinnere og
  gjøre alle gyldig for trekkning), er det bare å slette de to filene tidliger
  nevnt. 


  Brukernavn og passord for nettsiden ligger i - og kan endres i login.php i
  linjen: 'if ($brukernavn == "admin" && $passord == "hemmelig")'
